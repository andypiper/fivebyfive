Initial board definition for MicroPython

Tested with ESP-IDF 4.4 and MicroPython 1.18

### Usage

- Copy `01SPACE_C3_RGB` to `micropython/ports/esp32/boards`
- Run `make BOARD=01SPACE_C3_RGB`
